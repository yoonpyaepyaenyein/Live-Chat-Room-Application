# live-chat

# Chatting room website written in Vue.js, JavaScript, HTML & CSS


## Project setup
```
npm install
```

# serve with hot reload at localhost:8080
```
npm run serve
```
# serve with hot reload at localhost:3000
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
